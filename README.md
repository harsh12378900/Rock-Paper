# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Players can compete against the computer, view results instantly, and track their score in real time.

## 🚀 Features

- 🎮 User vs Computer Gameplay
- 🏆 Real-Time Score Tracking
- 🎲 Random Computer Choice Generation
- ⚡ Instant Win/Lose/Draw Results
- 🔄 Play Multiple Rounds
- 📱 Responsive User Interface
- 🎨 Clean and Interactive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 JavaScript Concepts Used

- DOM Manipulation
- Event Listeners
- Functions
- Conditional Statements
- Random Number Generation
- Variables and Data Types
- Dynamic Content Updates
- Game Logic Implementation

## 🎯 How to Play

1. Select Rock, Paper, or Scissors.
2. The computer randomly selects its choice.
3. The winner is determined based on the game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores are updated automatically.
5. Continue playing to achieve the highest score.
